YoungStar Club

Role

Lead Developer for building a responsive and interactive website to showcase club achievements and activities.

Key Tools Used

HTML5/CSS3 JavaScript Angular 4 Bootstrap PHP jQuery Adobe Illustrator Adobe Photoshop MAMP Server Sublime Text

Overview

YoungStar Club is a football club based in Pokhara, Nepal, established in 2010. The club aims to promote local talent and hosts national and local tournaments. It is also active in social activities, including blood donation drives and sanitation programs, making it a hub for both sports and community outreach.
