Galaxy Fire Project

Role

Game Developer and Programmer

Key Tools & Technologies

Unity 2017.2 C++ Autodesk Maya Adobe Illustrator Visual Studio

Project Overview

"Galaxy Fire" is a 3D single-player shooting game where players control a spaceship navigating through space while avoiding and shooting down asteroids. Developed using Unity and C++, the game features interactive gameplay, engaging audio effects, and smooth keyboard controls.

Storyboard

Backend (Development Process)

I started by creating a 3D model of the spaceship and asteroids in Autodesk Maya, followed by designing the background in Adobe Illustrator. These assets were imported into Unity, where I developed game objects such as the player, enemy, GUI text, and background. C++ scripts were used to implement real-time interactions and gameplay mechanics.

Front-End (User Experience)

The game features straightforward controls: use the arrow keys for navigation and the spacebar to shoot asteroids. Press 'R' to restart after a game over. The UI was designed to provide a easy gaming experience.
