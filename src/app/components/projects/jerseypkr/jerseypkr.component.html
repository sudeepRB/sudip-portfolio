JerseyPkr Project

Role

As a Web Developer Intern, my role encompassed designing logo, theme, wireframes, developing the user interface, and ensuring a smooth user experience for the JerseyPkr e-commerce platform.

Key Tools Used

Adobe Photoshop Balsamiq Adobe Dreamweaver Notepad ++ HTML CSS JavaScript

Project Overview

The JerseyPkr project involved developing an e-commerce platform that specializes in customizable sports jerseys. The project focused on creating an engaging, responsive interface with a user-friendly design that allows customers to navigate easily and customize their purchases.

Page Layout

The platform layout was designed to prioritize user convenience. Key elements include an intuitive homepage with clear navigation menus, a detailed product customization page, and a streamlined checkout process. The design also ensures consistency across devices, maintaining a high-quality user experience on both desktop and mobile.
